# Web - bootstrap4 - bs4

> Python3.7.6 Flask template

## Quick Start

```bash
# Install node
> sudo apt-get install node

# npm init o install
> npm install
> npm install --global gulp-cli

# Install dependencies npm
> npm install gulp browser-sync gulp-sass --save-dev
> npm install bootstrap jquery popper.js --save

# to start the dev
> gulp

```
